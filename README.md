JavaCrush is a single-player game inspired by the famous Candy Crush. The player's goal is to create matches - sequences of at least three items of the same type - by swapping adjacent cells on a grid to satisfy predetermined requirements at the start of the game, all within a limited number of moves.
To assist the player, the game features power-ups: special buttons that can be activated once per game to perform unique moves (for example, clearing an entire selected row). During gameplay, stalls might occur, which are situations where no available move leads to a match. In these cases, the elements on the grid are reshuffled to generate new, stall-free configurations.
The game can end in two possible outcomes: victory, if all requirements are met, or defeat, if the player runs out of moves.

# Instructions

Upon launch, the application displays a startup screen with two buttons: clicking "Play" opens the levels menu, while clicking "How to Play" opens the game instructions, which includes a button to return to the main menu.
Once inside the levels menu, simply select a level to start the game. Alternatively, clicking the "Menu" button in the bottom-left corner will return you to the startup screen.
During gameplay, to swap two cells, simply click on the first one and then on the second. If the two cells are not adjacent or do not result in a match, the swap will not occur. If you do not click a cell for 10 seconds, a few cells will illuminate to suggest a possible match.
To use a power-up, select one via the buttons on the right side of the game grid and then click on a specific cell. To cancel the selection of a power-up, simply click on it again or click on a different one.
While in a game, clicking the "Quit" button will close the current session and return you to the levels menu.
